# Screenshots and examples

This folder stores app screenshots and sample plots in a GitHub-friendly format (PNG recommended). All images from the previous gallery are consolidated here.

- Subfolders:
  - `app/`: app screens.
  - `plots/`: plot examples.
- Clear names:
  - `app_01_home.png`, `app_02_downloads.png`, `plot_boxplot.png`, `plot_correlation.png`.
- Format and size:
  - PNG recommended, under 2 MB per file for fast loading on GitHub.
- Privacy:
  - Do not include sensitive data in the screenshots.
